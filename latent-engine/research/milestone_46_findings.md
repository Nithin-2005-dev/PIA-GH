# Research Findings - Milestone 46

## Summary

M46 proves the platform can move from normalized observations to measurements, semantic evidence, and expertise estimates through a single coherent path.

## Findings

- The M40 measurement provenance already carries target entities, so semantic evidence can be mapped to modules or files.
- Actor attribution needed to be preserved in measurement metadata for expertise estimation.
- A bridge layer is safer than changing the legacy estimator contract directly.
- Semantic evidence can fan out to multiple estimator evidence records when several actors support the same target.

## Remaining Work

- Add probabilistic expertise updates.
- Add team and repository-level projections.
- Add confidence decomposition specific to expertise.
- Add persistent projection repositories.
- Add graph-aware estimation once the knowledge graph is durable.
